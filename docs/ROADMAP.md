# 덕뷰 고도화 계획

> 작성: 2026-05-20
> 단순 팬덤 링크 모음이 아니라, 직장인 / 4050 / 덕질 입문자도 **5분 안에 오늘 뭐가 중요한지 판단**할 수 있는 팬덤 브리핑 앱.

---

## 0. 제품 정의

**핵심 문장**
> 오늘 덕질에서 진짜 중요한 것만 정리해주는 앱

**사용자가 덕뷰에서 얻어야 하는 것**
- 오늘 뭐가 중요한지
- 이게 공식인지 추정인지 팬덤 반응인지
- 왜 중요한지
- 내가 뭘 하면 되는지
- 곧 놓치면 안 되는 일정이 뭔지

---

## 1. 핵심 타깃

### 4050 직장인 팬
- **Pain:** 시간 없음 / 트위터·X 안 함 / 커뮤니티 흐름 따라가기 어려움 / 용어 모름 / 루머·렉카 구분 힘듦
- **Want:** 5분 안에 오늘 중요한 것만 / 공식·추정 확실히 구분 / 일정·예매 놓치지 않기

### 덕질 입문자
- **Pain:** 컴백·직캠·단콘·음방 같은 용어 어려움 / 왜 이게 중요한지 맥락 부족 / 팬덤 분위기 파악 어려움
- **Want:** 입문자용 한 줄 설명 / 왜 화제인지 설명 / 공식·팬덤·뉴스 구분

### 공통 니즈
- **Pain:** 렉카·루머·스팸에 시간 낭비 / 일정 놓침 / 여러 플랫폼 돌아다니기 피곤함
- **Want:** 공식 vs 추정 vs 팬덤 반응 구분 / D-day / 오늘 할 일 / 놓친 소식 요약

---

## 2. 제품 원칙

### 유지할 원칙
- 로그인 없이 바로 볼 수 있어야 함
- 태민 MVP에서 먼저 검증
- 라이트 테마 중심
- 과한 팬시 UI보다 정보 위계 중심
- 링크 모음이 아니라 판단 도구
- 4050 / 입문자도 이해 가능한 문장
- 공식 / 추정 / 팬덤 반응을 분리
- 알림 / 프리미엄 / 광고는 기능 안정화 후

### 피해야 할 것
- 처음부터 로그인 강제
- 과한 glass / shadow / 떠오르는 카드
- 너무 많은 카드 나열
- 커뮤니티 느낌만 강한 UI
- 입문자 설명 과다 노출
- 전체 리팩토링
- 하단 네비 개편 먼저 하기

---

## 3. 현재 UI 방향

**라이트 UI 확정**
- 라이트 테마 유지 / 다크 fallback 가능성 유지
- 효과보다 정보 구조 중심
- Notion / Linear / 토스 / 최신 SaaS처럼 절제된 정보 피드 톤

**디자인 원칙**
- shadow보다 border / background / spacing으로 구분
- hover는 떠오름이 아니라 집중도 증가
- 제목은 명확하지만 과하지 않게
- meta는 조용하게
- 버튼은 튀지 않지만 클릭 가능하게
- 카드 중첩 최소화
- 섹션 안 항목은 리스트 / 여백 중심

## 4. 우선순위 로드맵

### P0 — 오늘 판단 기능

#### 1. Hero Briefing
가장 중요. 페이지 최상단에 **오늘 가장 중요한 1개 토픽**.

포함:
- 오늘의 핵심 헤드라인
- 관련 N건
- 출처 다양성
- 확정 / 추정 / 반응
- 왜 중요한지 한 줄
- 입문자 맥락
- 신뢰도

예:
> 태민 컴백 관련 소식이 여러 출처에서 반복되고 있어요
> 추정 · 팬덤 반응 증가 · 관련 7건

**목표:** 앱이 "오늘 뭐가 중요한지 판단해준다"는 인상을 첫 화면에서 바로 줌.

#### 2. 3줄 TL;DR
오늘 전체를 3줄로 정리.

- **공식 발표**
- **화제 이슈**
- **팬덤 분위기**

예:
> 공식: 오늘 공식 채널은 조용해요
> 화제: 컴백 관련 글이 반복되고 있어요
> 팬덤: 콘서트 후기와 직캠 반응이 많아요

카드 없는 경우도 솔직히 표시 — "오늘은 조용해요".

#### 3. 확정 / 추정 / 반응 구분
입문자와 4050에게 매우 중요. "이거 진짜야?"를 빠르게 판단 가능.

| 분류 | 의미 | 표현 |
|------|------|------|
| 확정 | 공식 / 언론 확인 | 확정 · 공식 채널 |
| 추정 | 정황은 있으나 공식 발표 전 | 추정 · 공식 발표 전 |
| 반응 | 팬덤에서 화제 | 반응 · 팬커뮤니티 중심 |
| 노이즈 | 신뢰 낮음 / 제외 후보 | (피드에서 숨김 또는 노이즈 리포트로 이동) |

### P1 — 중요도 기반 피드

#### 4. 3섹션 구조 (현재 방향 유지)
- 꼭 봐야 함
- 지금 화제
- 여유 되면

**각 카드:** 제목 / 출처 / 시간 / 왜 중요한지 한 줄 / 신뢰도·출처 유형

**목표:** 사용자가 중요도 순서대로 읽게 만들기.

#### 5. 왜 중요한지 한 줄
덕뷰의 핵심 가치. 모든 카드에 길게 붙이기보다, 중요한 카드에만 짧게.

예:
- 공식 발표라 놓치면 안 돼요
- 컴백 키워드가 반복되고 있어요
- 여러 출처에서 동시에 다뤄지고 있어요
- 팬덤 반응이 빠르게 늘고 있어요
- 입문자가 놓치기 쉬운 맥락이에요

#### 6. 출처 다양성 표시
단순 "관련 N건"보다 신뢰감 ↑.

예: `관련 7건 · 뉴스 3 · 팬덤 3 · 공식 1`

이걸로 파악 가능:
- 여러 곳에서 반복되는지
- 공식이 포함됐는지
- 팬덤만 떠드는 건지

---

### P1 — 일정 / 액션 기능

#### 7. 다가오는 일정 위젯 (현재 구현 방향 유지)
- 컴백 / 콘서트 / 팬미팅 / 음방 / 예매 시작
- D-day / 예매 알림 버튼

추가해야 할 것:
- 확정 / 예상 표시
- 공식 발표 대기 표시
- 캘린더 추가 후보

#### 8. 오늘 놓치면 안 되는 액션
방문율에 좋음. 읽기만 하는 앱이 아니라 "오늘 뭘 하면 되는지" 알려줌.

예:
- 공식 영상 보기
- 예매 일정 확인
- 컴백 날짜 저장
- 관련 기사 확인
- 팬덤 반응 보기

표현: `오늘 체크할 것 3개`

#### 9. 알림 / 리마인더
- **MVP (로그인 전):** 캘린더 추가 / Google Calendar 링크 / D-day 저장. 브라우저 알림은 후순위.
- **나중:** 푸시 알림 / 예매 1시간 전 알림 / 컴백 당일 알림.

### P1.5 — 재방문 기능

#### 10. 놓친 소식 요약
방문율 핵심. 직장인 / 4050에 매우 잘 맞음.

예:
- 지난 방문 이후 핵심 5개
- 어제 이후 놓친 소식
- 최근 7일 핵심 요약

localStorage로도 가능: 마지막 방문 시간 저장 → 이후 카드만 강조 / 요약 표시.

#### 11. 조용한 날 UX
팬덤 앱은 매일 이슈가 있지 않음. 조용한 날에도 쓸모 있어야 함.

예:
> 오늘 공식 소식은 조용해요
> 대신 최근 7일 중 이 소식은 챙겨보세요
> 다음 일정까지 D-61

#### 12. 팬덤 온도
감정적 재방문 요소. 매일 바뀌는 "날씨" 같은 기능.

예:
> 오늘 팬덤 온도: 🔥 높음
> 공식은 조용하지만 팬덤 반응은 활발해요
> 콘서트 후기 글이 늘고 있어요

#### 13. 오늘의 대표 영상
K-pop은 영상 소비가 강함.

- 오늘 많이 도는 영상
- 오늘의 직캠
- 공식 영상 먼저 보기
- 입문자 추천 영상

바쁜 사용자 니즈: "하나만 볼 거 추천해줘"

### P2 — 개인화 / 입문자 기능

#### 14. 입문자 모드
- **ON:** 용어 설명 / 맥락 설명 / 공식·추정 구분 강화
- **OFF:** 코어 팬용 간결 모드

주의:
- 설명이 너무 많으면 화면이 무거워짐
- 기본은 한 줄
- 긴 설명은 접기 / 툴팁

#### 15. 관심사 태그
처음엔 localStorage로 가능.

예: 컴백 / 콘서트 / 예매 / 직캠 / 공식 영상 / 팬덤 반응

나중에 알림 / 프리미엄으로 확장 가능.

#### 16. 노이즈 리포트
덕뷰 차별점. 4050 / 입문자에게 강한 가치.

예:
- 오늘 제외한 노이즈 12건
- 공식 확인 안 된 루머 3건 숨김
- 렉카성 제목 필터링됨

효과: "내가 직접 안 걸러도 된다"는 신뢰감.

---

### P3 — 로그인 / 프리미엄

#### 17. 로그인
초기 MVP에서는 필요 없음.

**초기:**
- 로그인 없이 바로 사용
- localStorage 기반 개인화 (마지막 방문 시간 / 관심사 / 읽음 표시 / 입문자 모드)

**나중에 필요한 순간:**
- 기기 간 동기화 / 푸시 알림 / 프리미엄 / 여러 아티스트 관리 / 내 덕질 로그

**선택 로그인 후보:** Google / Apple / Kakao / 이메일 매직링크

#### 18. 프리미엄 (기능 안정화 후)
후보:
- 무료 관심 아티스트 3명 / 프리미엄 무제한
- 광고 제거
- 고급 알림
- 최근 30일 요약
- 다중 아티스트 캘린더
